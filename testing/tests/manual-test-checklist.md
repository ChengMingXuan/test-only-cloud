#### ☐ CHG-ORDER-008：订单费率跨时段计算验证

**配置前置**：
```
费率配置：
  尖峰（10:00-12:00）：1.2 元/kWh
  高峰（12:00-14:00）：0.85 元/kWh
  平段（14:00-22:00）：0.55 元/kWh
  谷段（22:00-06:00）：0.25 元/kWh
  
场站已绑定该费率
```

**操作步骤**：
1. 在模拟器中启动一个设备
2. 启动充电，时间设置为 09:50（跨越时段）
3. 设置充电至 12:10（跨越三个时段）
4. 充电完成，查看订单详情中的费用计算

**预期结果**：
- [ ] 费用计算按照分时时段进行：
  - 09:50-10:00：无电价（或谷段价格）
  - 10:00-12:00：10:00-12:00 的电量 × 1.2 元/kWh
  - 12:00-12:10：12:00-12:10 的电量 × 0.85 元/kWh
- [ ] 总金额 = Σ(分段电量 × 对应电价) + 服务费 - 优惠
- [ ] 费用精确到分（2 位小数），无舍入误差

**举例验证**：
```
假设：
  09:50-10:00 充 0.2 kWh（谷段 0.25）= 0.05 元
  10:00-12:00 充 5 kWh（尖峰 1.2）= 6 元
  12:00-12:10 充 1.5 kWh（高峰 0.85）= 1.275 元
  服务费 = 6.7 × 0.08 = 0.536 元
  总金额 = 0.05 + 6 + 1.275 + 0.536 ≈ 7.86 元
```

**验证通过**：☐ 是 ☐ 否
