```powershell
if (-not (Test-DatabaseConnection)) {
    Write-Log "❌ 无法连接数据库，停止执行" "Red"
    exit 1
}

# 即使迁移有警告，也继续执行
if ($LASTEXITCODE -eq 0 -or $LASTEXITCODE -eq 1) {
    Write-Log "✅ 成功" "Green"
}
```

### 3. **测试隔离和顺序执行**

```powershell
foreach ($test in $tests) {

}
```
