### 5分钟快速安装

#### 第一步：安装油猴脚本（2分钟）

1. **安装Tampermonkey扩展**
   - Chrome: https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo
   - Firefox: https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/

2. **导入脚本**
   - 打开Tampermonkey管理面板 → 实用工具 → 从文件导入
   - 导入 `manual-test-helpers/tampermonkey/aiops-form-autofill.user.js`
   - 导入 `manual-test-helpers/tampermonkey/aiops-test-toolbar.user.js`

#### 第二步：安装Playwright半自动（2分钟）

```powershell
cd tests/manual-test-helpers/playwright-semi-auto
npm install
```

#### 第三步：验证（30秒）

1. **验证油猴脚本**
   - 访问 https://aiops.jgsy.com
   - 右上角应显示测试工具栏
   - 右下角应显示3个浮动按钮（智能填充/清空/随机）
   - 按 `Alt+T` 可隐藏/显示工具栏
   - 按 `Alt+F` 可自动填充表单

2. **验证Playwright半自动**
   ```powershell
   npm run test:charging
   ```
   - 浏览器自动打开
   - 暂停点等待你按Enter
   - 测试流程引导执行

### 方式二：命令行手动执行

### 1. 安装依赖（5分钟）

```bash
# 根目录执行
cd tests

# pytest API测试
cd test-automation && pip install -r requirements.txt && cd ..

# Playwright E2E
cd playwright-tests && npm install && npx playwright install --with-deps && cd ..

# Selenium兼容性
cd selenium-tests && pip install -r requirements.txt && cd ..

# Puppeteer性能
cd puppeteer-tests && npm install && cd ..

# Cypress组件
cd cypress-tests && npm install && cd ..

# 统一编排器
cd test-orchestrator && npm install && cd ..
```

### 2. 首次测试（10分钟）

```bash
# 单独运行Playwright冒烟测试
cd playwright-tests
npm run test:smoke

# 单独运行pytest
cd ../test-automation
pytest tests/test_auth.py -v

# 运行Cypress UI模式（可视化）
cd ../cypress-tests
npm run cy:open
```

### 3. 完整测试（4-5小时）

```bash
# 使用统一编排器执行全部测试
cd test-orchestrator
npm run test:all

# 查看聚合报告
start ../test-reports/aggregated-report.json
```
