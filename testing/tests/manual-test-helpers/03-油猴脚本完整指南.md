## ⚡ 快速开始（3 分钟）

### 什么是油猴脚本？

油猴脚本是在浏览器中**实时注入数据和自动操作**的脚本。特点：

- ⚡ **超快**：5-15 秒内完成一个验证
- 🎯 **精准**：针对特定功能的快速检查
- 🚀 **即时反馈**：开发完成立刻验证
- 📱 **轻量**：无需编程知识即可使用

### 第一步：安装浏览器扩展

1. 打开 Chrome/Firefox 浏览器
2. 搜索"Tampermonkey"并安装官方扩展
3. 刷新页面，扩展会自动激活

### 第二步：安装我们的油猴脚本

每个脚本都可以直接在浏览器中安装：

```
1. 打开项目目录：tests/tampermonkey-scripts/
2. 打开任意 .js 脚本文件
3. 复制全部代码
4. 在 Tampermonkey 菜单 → "添加新脚本"
5. 粘贴代码，保存
6. 刷新系统页面
```

### 第三步：使用脚本

1. 在系统页面上，点击浏览器右上角的 **Tampermonkey 图标** 🔧
2. 在下拉菜单中看到我们的所有脚本
3. 点击对应的脚本名称
4. 脚本自动执行（5-15 秒）
5. 查看控制台输出结果

### 故障排查

#### 问题 1：脚本没有出现在菜单中

```
现象：点击 Tampermonkey 图标后没有看到脚本

可能原因：
  1. 脚本没有保存
  2. 脚本的 @match 规则不匹配当前页面
  3. 脚本被禁用

解决方法：
  1. 确认脚本已保存（Ctrl+S）
  2. 检查 @match 规则：
     当前地址：https://localhost:5000/user/create
     规则应该是：@match https://localhost:5000/*
  3. 在 Tampermonkey "管理脚本"中启用脚本
     └─ 找到脚本，点击复选框启用
```

#### 问题 2：脚本运行但没有效果

```
现象：脚本菜单中有，点击了，但页面没有反应

可能原因：
  1. 页面还没加载完成
  2. 表单元素类名/ID 改变了
  3. API 端点改变了

解决方法：
  1. 等待页面完全加载后再运行脚本
  2. 打开 F12 → 控制台，查看错误信息
     例如："Cannot find element with ID 'userNameInput'"
     → 说明表单元素已改变，需要更新脚本
  3. 检查网络标签页，看 API 请求是否成功
     └─ 如果红色，表示请求失败
     └─ 右键检查响应内容查看具体错误
```

#### 问题 3：脚本执行很慢

```
现象：脚本运行时间超过预期

可能原因：
  1. 网络延迟
  2. 系统负载过高
  3. 脚本内有过长的等待时间

解决方法：
  1. 检查网络延迟：
     └─ F12 → Network 标签页
     └─ 观察请求响应时间（应该 < 500ms）
  2. 重启浏览器和系统
  3. 在非高峰期运行脚本
```

#### 问题 4：表单提交失败

```
现象：脚本运行后，API 返回 400 或 500 错误

可能原因：
  1. 生成的数据不符合要求（如邮箱格式）
  2. API 验证规则改变了
  3. 后端服务异常

解决方法：
  1. 查看控制台错误信息，获取具体原因
  2. 手动填充表单，确认什么数据有效
  3. 更新脚本中的数据生成逻辑
  4. 如果是后端问题，联系开发人员
```

```javascript
// user-create.js 文件的核心逻辑：

// 1. 生成随机数据
const timestamp = new Date().getTime();
const username = `test_user_${timestamp}`;
const email = `${username}@example.com`;

// 2. 查找页面上的表单元素
const usernameField = document.getElementById('username');
const emailField = document.getElementById('email');
const submitButton = document.querySelector('button[type="submit"]');

// 3. 填充表单
usernameField.value = username;
emailField.value = email;

// 4. 触发表单验证事件
usernameField.dispatchEvent(new Event('change'));
emailField.dispatchEvent(new Event('change'));

// 5. 点击提交
submitButton.click();
```
