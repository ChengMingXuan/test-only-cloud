## 测试执行指南

### 快速开始

```bash
# 进入项目目录
cd tests/manual-test-helpers/playwright-semi-auto

# 安装依赖
npm install && npx playwright install

# 运行能源专项测试
npm run scenario:energy-core         # 能源核心基础设施
npm run scenario:energy-trading      # 能源交易与商业
npm run scenario:energy-ops          # 能源运维与多元服务
npm run scenario:energy-all          # 运行全部能源场景
```

### 执行要求

1. **环境准备**：
   - Node.js ≥ 18.0.0
   - Playwright 浏览器已安装
   - 测试系统已启动并可访问
   - 管理员账户已配置

2. **测试人员**：
   - 需要至少 2 小时的完整时间
   - 熟悉能源管理领域基本概念
   - 能够理解和执行测试步骤中的业务逻辑
   - 准备好截图和反馈

3. **监控指标**：
   - 每个步骤的执行时间
