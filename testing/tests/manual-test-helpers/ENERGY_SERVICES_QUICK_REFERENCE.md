## 🎯 快速开始（5 分钟）

### 步骤 1：进入项目目录

```bash
cd tests/playwright-scenarios
```

### 步骤 2：选择运行方式

**方式 A：一次运行全部 3 个场景（推荐，耗时 120-180 分钟）**

```bash
npm run scenario:energy-all
```

**方式 B：逐个运行（推荐用于调试，可分步骤）**

```bash
npm run scenario:14          # 能源基础设施
npm run scenario:15          # 能源交易
npm run scenario:16          # 能源运维
```

### 步骤 3：按照系统提示进行测试

每个步骤系统会：
1. 自动导航和设置
2. 提示"请验证 XXX"
3. 等待你按 Enter 或输入 ❌

### 步骤 4：查看报告

```bash
# 生成 HTML 报告
npx playwright show-report
```

## 🚨 快速故障排查

### 常见问题（按出现频率）

#### 问题 1：某个 Scenario 超时

**现象**：运行了 30+ 分钟某个步骤还没完成  
**原因**：网络延迟或后端响应慢  
**解决**：
```bash
# 查看后端日志
docker-compose logs -f jgsy-energycore-microgrid

# 如果是持续超时，可能需要优化数据库查询
# 联系能源模块的开发人员
```

#### 问题 2：测试步骤失败，有错误提示

**现象**：看到 ❌ 标记和错误信息（如"微网配置失败"）  
**原因**：功能异常或数据错误  
**解决**：
1. 查看浏览器截图（报告中有）
2. 查看 API 返回的错误信息
3. 检查测试数据是否有效
4. 运行单个失败的场景进行深度调试

#### 问题 3：报告无输出或很少

**现象**：测试完成但报告空或只有 1-2 步  
**原因**：脚本提前退出或未正确保存  
**解决**：
```bash
# 查看测试输出
npm run scenario:14 -- --reporter=list

# 如果看到异常，查看详细日志
npm run scenario:14 -- --debug
```
